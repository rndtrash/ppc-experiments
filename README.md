# Hellorld Open Firmware

Based on [thamugadi/powerpc-ofw-boot](https://github.com/thamugadi/powerpc-ofw-boot)

# Building

 1. Install kpartx, hfsplus tools with mkfs.hfsplus program and GCC with Binutils for PPC
 2. Run `make`
